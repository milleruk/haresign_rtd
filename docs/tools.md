# Tools

Haresign tools are grouped around common primary care management questions:
funding, workforce, demand, population, access, quality, and assurance.

## Funding and Finance

| Tool | Purpose |
|---|---|
| ARRS Calculator | Estimate Additional Roles Reimbursement Scheme funding and role-level allocations. |
| PCN Funding Split | Model how PCN funding can be distributed across member practices. |
| Payroll Forecaster | Forecast payroll cost pressures and recurring staffing costs. |
| QOF Income Calculator | Estimate QOF income using practice list size and achievement assumptions. |
| IIF Calculator | Support Investment and Impact Fund planning and scenario modelling. |
| GP Reimbursement Calculator | Explore reimbursement assumptions for GP roles and claims. |
| NHS Payments Dashboard | Benchmark published NHS payments to practices and PCNs. |

## Access and Demand

| Tool | Purpose |
|---|---|
| GP Appointments Dashboard | Explore monthly GP appointment volumes, modes, wait bands, DNA rates, and capacity indicators. |
| GPAD Analyser | Upload and analyse local GP appointment extracts for mapping, pressure, and data-quality insights. |
| Online Consultations Dashboard | Explore online consultation usage by practice, PCN, supplier, and time pattern. |
| Cloud Telephony Dashboard | Explore call volumes, answer patterns, wait distributions, and telephony demand. |
| Demand Forecaster | Review demand trends and forecast planning indicators. |

## Population and Workforce

| Tool | Purpose |
|---|---|
| List Size Dashboards | Explore registered patient population, age bands, sex breakdowns, and geography. |
| NWRS Workforce Dashboard | Explore published workforce snapshots and staff mix. |
| Practice Overview | Bring together key public datasets for a selected practice. |
| PCN DES Dashboard | Review PCN DES achievement and indicator performance. |

## Quality, Experience, and Assurance

| Tool | Purpose |
|---|---|
| QOF Prevalence Dashboard | Compare disease register prevalence across practices and national benchmarks. |
| GP Patient Survey Dashboard | Review patient experience measures from the GP Patient Survey. |
| GP Contract Dashboard | Explore GP Contract Services and vaccination-related indicators. |
| CQC Searches | Support CQC evidence preparation and search planning. |
| Enhanced Access Tool | Support enhanced access planning and assurance. |

## Public Data Explorer

| Tool | Purpose |
|---|---|
| Fingertips NHS Profiles | Explore public OHID Fingertips profiles and indicators. |

## Developer API Coverage

Several datasets behind the tools are available through the developer API,
including practices, PCNs, population data, QOF, NWRS, online consultations,
list sizes, GP appointments, GP Patient Survey, NHS payments, cloud telephony,
PCN DES, and GP Contract data.

See [Developer API](api.md) for endpoints and examples.
