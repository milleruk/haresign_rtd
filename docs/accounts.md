# Accounts and Workspaces

You can use many Haresign tools without an account. Creating an account adds
features that make repeated use easier, especially if you regularly work with
the same practice or PCN.

## What An Account Adds

| Feature | What it helps with |
|---|---|
| Email verification | Confirms that account access belongs to you. |
| Practice linking | Associate your account with one or more GP practices. |
| PCN linking | Associate your account with one or more PCNs. |
| Workspace switching | Move between linked practice and PCN contexts. |
| Saved presets | Store common tool settings or selected organisations. |
| My documents | Manage documents you have uploaded or shared through the document library. |

## Creating An Account

1. Go to `https://haresign.net/accounts/register/`.
2. Enter your details and create a password.
3. Check your email for the verification link.
4. Open the verification link to activate the account.
5. Sign in at `https://haresign.net/accounts/login/`.

If the verification email does not arrive, use the resend option on the
verification page.

## Profile Page

After logging in, your profile page is available at:

```text
https://haresign.net/accounts/profile/
```

Use it to:

- review your account details;
- link a practice;
- link a PCN;
- switch active workspace;
- manage membership requests;
- access your uploaded documents;
- delete your account if needed.

## Linking A Practice

1. Open your account profile.
2. Choose the option to link a practice.
3. Search by practice name or ODS code.
4. Select the correct practice.
5. Submit the access request.

If the practice already has an account admin, the request can be approved or
rejected by that admin. If not, platform administrators may need to review it.

## Linking A PCN

1. Open your account profile.
2. Choose the option to link a PCN.
3. Search by PCN name or code.
4. Select the correct PCN.
5. Submit the access request.

PCN membership works in the same way as practice membership: requests may need
approval before the PCN appears as an active workspace.

## Workspace Switching

A workspace is the practice or PCN context the site should use where a tool can
benefit from one.

For example:

- a practice workspace can pre-select the practice in practice-level tools;
- a PCN workspace can pre-select the PCN in PCN-level dashboards;
- saved presets can be scoped to the active context.

If you work across several organisations, check the active workspace before
saving presets or interpreting context-specific results.

## Membership Roles

Linked practices and PCNs can have different membership states.

| State | Meaning |
|---|---|
| Pending | You have requested access and are waiting for approval. |
| Active | You can use the practice or PCN as a workspace. |
| Admin | You can approve, reject, or remove other members for that organisation. |
| Rejected or removed | Access is not active. You may need to request access again. |

## Saved Tool Presets

Some tools support saved presets. A preset stores useful selections so you can
return to a common scenario or organisation quickly.

Typical preset examples:

- a frequently used practice;
- a PCN funding scenario;
- a reimbursement scenario;
- a calculator setup for board reporting.

Presets are intended to save time. They do not freeze the underlying source
data, so results can change when public datasets are refreshed.

## Password Reset

If you cannot sign in, use:

```text
https://haresign.net/accounts/password-reset/
```

You will receive a reset email if the address matches an account.

## Account Deletion

You can request account deletion from your profile page. Deleting an account
removes your account access. Documents you have uploaded may remain available if
they were already shared, but they will no longer be associated with your
account.
