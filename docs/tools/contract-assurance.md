# Contract and Assurance Tools

Haresign includes tools that support contractual review, reimbursement
planning, access planning, and assurance preparation.

## GP Contract Dashboard

Use the GP Contract Dashboard to explore GP Contract Services and
vaccination-related indicators.

It can show:

- practice-level indicator values;
- numerator and denominator measures where published;
- vaccination programme activity;
- geography comparisons;
- carers demographic breakdowns where available.

Always read the source indicator definition before comparing values.

## GP Reimbursement Calculator

Use this calculator to explore reimbursement scheme assumptions.

It can show:

- practice-level cap where available;
- route-specific estimates;
- adjusted population or weighting effects where relevant;
- prompts for eligibility checks.

Use it for planning support. Verify entitlement before submission.

## IIF Calculator

Use the IIF calculator for PCN incentive scenario planning.

Because IIF rules can change, confirm the current framework before using
outputs in formal planning.

## Enhanced Access Tool

Use the Enhanced Access Tool to support planning and review of enhanced access
provision.

Typical use:

1. Enter or review planned provision.
2. Check coverage and assumptions.
3. Export or record outputs for local discussion.
4. Compare with local commissioner requirements.

## CQC Readiness Tools

CQC readiness tools help structure evidence and preparation.

They can support:

- quality statement review;
- checklist-style evidence tracking;
- action planning;
- internal assurance discussions.

They do not replace CQC guidance, inspection judgement, or local governance.
