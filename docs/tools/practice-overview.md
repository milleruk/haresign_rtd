# Practice Overview

Practice Overview is the fastest way to understand a selected GP practice in
context. It brings together organisation identity, population, and public NHS
benchmark signals into one starting point.

## When To Use It

Use Practice Overview when you need to:

- prepare for a practice support conversation;
- understand a practice before looking at a detailed dashboard;
- check PCN, ICB, and regional context;
- spot areas that need deeper review;
- create a short briefing for internal discussion.

## What It Shows

The exact sections can vary as public datasets are refreshed, but the overview
is designed to bring together:

- practice identity and ODS code;
- PCN and geography context;
- registered population and age profile;
- access and demand signals;
- quality, survey, workforce, and payment context where available;
- links into more detailed dashboards.

## How To Use It

1. Search for the practice by name or ODS code.
2. Confirm that the selected practice is the intended organisation.
3. Review the population and geography context first.
4. Scan each headline section for unusual values or missing data.
5. Open the linked dashboards for sections that need more investigation.

## Good Questions To Ask

- Is the practice population growing, shrinking, or ageing?
- Is the comparison group appropriate?
- Are access indicators consistent across appointments, telephony, and survey
  data?
- Are payment or workforce differences explainable by contract type, services,
  premises, or local context?
- Which findings need local validation before action?

## Interpretation Notes

Practice Overview combines datasets with different reporting periods. Treat it
as a triage and navigation page, not a definitive performance score.

If a figure is used in a formal paper, cite the underlying source dataset and
reporting period.
